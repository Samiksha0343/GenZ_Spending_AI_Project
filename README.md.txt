# Gen-Z Spending Behavior & Money Personality Analysis

## Description
This project analyzes Gen-Z spending behavior using data analytics and AI.
Machine learning is used to classify users into money personalities.

## Technologies Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-learn

## Features
- Spending analysis
- Money personality classification
- AI-based recommendations
- Prediction for new users
